-----Windows PC スリープしないのロボット-----

■機能説明
　詳細内容  :このコードは、Robotクラスを使って、指定時間ごとで
　　　　　　　Windowsキーを繰り返し自動押下し、アプリもアクティブする。
　設定時間  :分単位 (整数型：非必須)  ---初期値[4]分--- 
　停止方法  :[CTRL + C ] キー同時に押下すると、処理終了する。
　
■実行環境
Java インストール済み環境で実行必要です。
PATH 環境変数は、 java.exeファイルが含まれている Java インストール済み環境のディレクトリーを指している必要があります。 
PATH に Java インストール済み環境の\bin ディレクトリーが含まれていることを確認します。

■実行方法
①バッチファイル「PressWinKey.bat」 をダブルクリックする。

②DOS画面でJavaクラスを起動する。
1.dos コマンドで、ファイル保存のディレクトリに移動します
例：cd C:\noSleep

2.javaファイルをコンパイルする。
 javac PressWinKey.java
 
3.処理を起動する （初期値 4 分）
 ※パラメータにすきな繰り返し時間を設定してください。
 java PressWinKey [パラメータ]
 例：java PressWinKey 4
 
■停止方法
　[CTRL + C ] キー同時に押下すると、スクリプトを停止する。
